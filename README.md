# Personal Website

## Local Development Server

To view the website locally, use one of these methods:

### Method 1: Using the Python server script (Recommended)

```bash
python3 server.py
```

The server will start at **http://localhost:8000**

### Method 2: Using Python's built-in server directly

```bash
python3 -m http.server 8000
```

### Method 3: Using the start script

```bash
./start.sh
```

Then open your browser and navigate to: **http://localhost:8000**

To stop the server, press `Ctrl+C` in the terminal.

---

## Quick Start

1. Start the server: `python3 server.py`
2. Open browser: http://localhost:8000
3. Edit files and refresh browser to see changes
